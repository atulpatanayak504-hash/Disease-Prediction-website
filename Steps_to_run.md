Here are the steps in simple language:

---

**What you need first**
Make sure Python is installed on your computer. You can check by typing `python --version` in your terminal or command prompt.

---

**Step 1 — Open the project folder**
Unzip the downloaded file. Open your terminal (or Command Prompt on Windows) and go into the folder:
```bash
cd disease_prediction
```

**Step 2 — Install the required libraries**
This installs everything the project needs to run:
```bash
pip install -r requirements.txt
```
Wait for it to finish. It only takes a minute.

**Step 3 — Start the website**
Run this command:
```bash
python app.py
```
You will see a message like `Running on http://127.0.0.1:5000` — that means it is working.

**Step 4 — Open the website**
Open any browser (Chrome, Firefox, Edge) and type this in the address bar:
```
http://127.0.0.1:5000
```
Your website will open.

**Step 5 — Use it**
Tick the symptoms you have and click the **Predict Disease** button. The result will appear on the screen.

---

**To close the website**, go back to the terminal and press `Ctrl + C`.
