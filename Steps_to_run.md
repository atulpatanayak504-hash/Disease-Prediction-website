STEPS TO RUN THE PROJECT:- 
Step-1:- Make sure all the required modules are download according to requirement.txt.
         (pip install -r requirements.txt) paste this in the vs code terminal to download or
         (py -m pip install -r requirements.txt) paste this if pip not recognized

Step-2:- After downloading the requirements.txt, run the app.py
Step 3:- It takes a liitle time and in the terminal you will see a http://127.0.0.1:5000
Step 4:- Use ctrl+click and it will direct you to the website.

This is the steps to run the project
